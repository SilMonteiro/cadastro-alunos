# Trabalho: Sistema de cadastro de alunos
Projeto final da disciplina de PRINCÍPIOS DE CONSTRUÇÃO DE ALGORITMOS (PCA) cujo o objetivo é criar um sistema de cadastro de alunos em python utilizando pandas.

## Funcionalidades do sistema
As principais funcionalidades desse sistema incluem:
- Cadastrar um novo aluno
- Editar dados dos alunos
- Remover algum cadastro

## Antes de executar o programa
Antes de utilizar será **necessário** instalar a biblioteca pandas utilizando o comando `pip intall pandas`

## Como executar o programa
Abra o terminal, utilize o comando `cd` para indicar o caminho do script e utilize o comando `python cadastrodosalunos.py`

## Tecnologia utilizada
- Biblioteca pandas
